In this post I compiled some data about Cloud providers GPU offerings: CPU and GPU performance, costs. etc. Beware that performance data is a rough estimate, and actual perfomance may vary grately between applications.

Using the graphs below you can compare estimate CPU and GPU performance, comapre costs and cost/performance values for a particular period. To select the period click on the "Cost per period" graph or use one of the buttons below it. You can also filter offerings by GPU numbers and providers.

You can find details for all offerings in the table at the bottom of this post. The data will be updated on (I hope) a regular basis.
