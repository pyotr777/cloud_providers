In this post I compiled some data about Cloud providers GPU offerings: CPU and GPU performance**, costs*, memory, etc.

Using the graphs below you can compare estimate CPU and GPU performance, compare costs and cost/performance values for a particular period. To select the period click on the "Cost per period" graph or use one of the buttons below it. You can also filter offerings by number of GPUs and by providers.

You can find details for all offerings in the table at the bottom of this post. The data will be updated on (I hope) a regular basis.

The below graph shows rent cost for a period. X-axis is rent period, Y-axis is rent cost.

All offers are charged in time units: hours, weeks, months or years. If an offer is charged by months, for example, using it for 1 day will cost you monthly price. Thus, the below graph can be thought of as the minimum sum you have to pay for using offers for particular period.

Clicking on the graph will change graphs below it to display costs for the selected period. You can also set the period with the buttons below.
